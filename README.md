> *Making large models run on hardware you can actually own.*

My core research is **SAAQ** (Spiking Adaptive Activity Quantization) — a method for compressing large-scale models to run on consumer GPU hardware by converting them into spiking neural network representations. I also work on model interpretability and benchmarking quantization quality.
